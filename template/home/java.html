1. Java Introduction

Java is a popular programming language that is widely used for building applications. It is object-oriented, platform-independent, and known for its portability across platforms.

Java code is written once and can run on any device that supports Java without the need for recompilation. This is made possible through the Java Virtual Machine (JVM).

2. Getting Started with Java

To start coding in Java, you need to install the Java Development Kit (JDK). Here’s how:

  1. Download the JDK from the official Oracle website.
  2. Install the JDK following the instructions for your operating system.
  3. Verify the installation by typing java -version in your command prompt.

3. Java Syntax

Java follows a specific syntax and structure. Every Java program begins with a class declaration, which must match the file name. The main method inside the class serves as the entry point for program execution.

Here’s a simple example of a Java program that prints "Hello, World!" to the console:


          public class HelloWorld {
              public static void main(String[] args) {
                  System.out.println("Hello, World!");
              }
          }

public class HelloWorld { ... }

public: This keyword makes the class accessible from anywhere, so other parts of the program can access it if needed.

class: This keyword is used to declare a new class, which is like a blueprint for creating objects or organizing code.

HelloWorld: This is the name of the class. By convention, class names start with a capital letter. In this example, the class is named HelloWorld because it represents the code for a simple "Hello, World!" program.

The entire program is inside this HelloWorld class.

public static void main(String[] args) { ... }

public: This means the method is accessible from anywhere in the program.

static: This keyword means the method belongs to the class itself and can be run without creating an instance (object) of the class.

void: This indicates that the method does not return any value.

main: The main method is the entry point of any Java application. When you run the program, Java starts executing from this method.

String[] args: This is an array of String values that can hold command-line arguments if you pass any when starting the program.

Another example:


          public class Greeting {
              public static void main(String[] args) {
                  System.out.println("Welcome to Java programming!");
              }
          }

And one more example:


          public class Addition {
              public static void main(String[] args) {
                  int a = 5;
                  int b = 10;
                  System.out.println("Sum is: " + (a + b));
              }
          }

4. Java Output

Java provides several methods to display output to the console. The most common method is System.out.println(), which prints the output followed by a new line.

In Java, "output" refers to displaying information to the user or console. Java provides a way to produce output using the System.out.print() and System.out.println() methods, both part of the System class in Java’s java.lang package.

Here’s an example:


          System.out.println("Hello, Java!");

Another method is System.out.print(), which does not add a new line:


          System.out.print("This is on the same line.");

Here’s a combined example:


          System.out.print("Welcome to ");
          System.out.println("Java programming!");

5. Java Comments

Comments in Java are non-executable lines that are used to provide explanations within the code. Java supports three types of comments: single-line comments, multi-line comments, and documentation comments.

Example of a single-line comment:


          // This is a single-line comment

Example of a multi-line comment:


          /*
          This is a multi-line comment.
          It can span multiple lines.
          */

Example of a documentation comment:


          /**
           * This is a documentation comment.
           * It is used to generate API documentation.
           */

Another example:

6. Java Variables

Variables in Java are containers for storing data values. Each variable must have a specific data type, which determines the kind of data it can hold.

Here’s how to declare and initialize variables:


          int age = 30;
          String name = "John";

int age = 30;

int: This keyword is a data type in Java that stands for "integer." It’s used to store whole numbers (without decimal points), such as -5, 0, or 30.

age: This is the variable name. In this case, it represents a person’s age, which is why we named it age. You can choose other names for variables, but it’s good practice to pick names that make sense for what they store.

= 30;: The equals sign (=) is the assignment operator in Java. It assigns the value on the right side (30) to the variable on the left side (age). The semicolon (;) ends the statement.

So, int age = 30; declares an integer variable called age and assigns it the value 30.

String name = "John";

String: This is a data type in Java used to store text (a sequence of characters). In Java, String values are always enclosed in double quotes (").

name: This is the variable name, representing the name of a person.

= "John";: The equals sign assigns the value "John" to the variable name. The semicolon ends the statement.

So, String name = "John"; declares a String variable called name and assigns it the value "John".

Another example:


          double price = 19.99;
          boolean isAvailable = true;

double price = 19.99;

double: This is a data type in Java used to store decimal numbers, such as 19.99 or 3.14. It’s commonly used for values that may contain fractions.

price: This is the variable name. In this case, it represents the price of an item, so it’s named price to make its purpose clear.

= 19.99;: The equals sign (=) is the assignment operator in Java. It assigns the value on the right side (19.99) to the variable on the left side (price). The semicolon (;) ends the statement.

So, double price = 19.99; declares a decimal variable called price and assigns it the value 19.99.

boolean isAvailable = true;

boolean: This is a data type in Java used to store true or false values. It’s often used for conditions, such as checking if something is available.

isAvailable: This is the variable name, chosen to represent whether an item is available. The name isAvailable is descriptive of what it stores.

= true;: The equals sign assigns the value true to the variable isAvailable. The semicolon ends the statement.

So, boolean isAvailable = true; declares a boolean variable called isAvailable and assigns it the value true.

One more example:


          char initial = 'J';
          long population = 7000000;

char initial = 'J';

char: This is a data type in Java used to store a single character, such as a letter or symbol. In Java, characters are enclosed in single quotes (').

initial: This is the variable name, which represents the initial of a person’s name. It’s named initial to clearly indicate what it holds.

= 'J';: The equals sign (=) is the assignment operator in Java. It assigns the value on the right side ('J') to the variable on the left side (initial). The semicolon (;) ends the statement.

So, char initial = 'J'; declares a character variable called initial and assigns it the value 'J'.

long population = 7000000;

long: This is a data type in Java used to store large integer values, which can hold numbers much larger than the int data type. It's useful for values like populations or large counts.

population: This is the variable name, representing the population of a place. It's named population to clearly describe what it stores.

= 7000000;: The equals sign assigns the value 7000000 to the variable population. The semicolon ends the statement.

So, long population = 7000000; declares a long variable called population and assigns it the value 7000000.

7. Java Data Types

In Java, data types specify the kind of values a variable can hold. Java is a strongly-typed language, meaning each variable must be declared with a data type before it’s used. Java's data types are divided into two main categories:

Primitive Data Types: The most basic data types in Java.

Non-Primitive (Reference) Data Types: More complex data types, like classes, arrays, and interfaces.

Java two categories of data types: primitive types and reference types. Primitive types include int, char, double, etc.

Example of using primitive data types:


          int number = 10;
          char letter = 'A';
          double price = 10.99;

Example of a reference data type:


          String message = "Hello, World!";

String message = "Hello, World!";

String: This is a data type in Java used to store text (a sequence of characters). In Java, String values are always enclosed in double quotes (").

message: This is the variable name. In this case, it represents a message or text. It’s named message to clearly indicate what it holds.

= "Hello, World!";: The equals sign (=) is the assignment operator in Java. It assigns the value on the right side ("Hello, World!") to the variable on the left side (message). The semicolon (;) ends the statement.

So, String message = "Hello, World!"; declares a String variable called message and assigns it the value "Hello, World!".

And another example of a reference data type:


          String greeting = "Good Morning!";

8. Java Type Casting

Type casting in Java is the process of converting a variable from one data type to another. There are two types of casting: widening and narrowing.

Example of widening casting:


          int num = 100;
          double dNum = num; // Widening

Example of narrowing casting:


          double dNum = 9.78;
          int num = (int) dNum; // Narrowing

Another example of narrowing casting:


          double decimalNum = 5.99;
          int wholeNum = (int) decimalNum; // Will result in 5

9. Java Operators

Java has several types of operators: arithmetic, relational, logical, assignment, bitwise, and more. Operators are special symbols that perform operations on variables and values.

Example of arithmetic operators:


        int a = 10;
        int b = 5;
        int sum = a + b; // Addition
        int difference = a - b; // Subtraction
        int product = a * b; // Multiplication
        int quotient = a / b; // Division
        int remainder = a % b; // Modulus

Example of relational operators:


        boolean isEqual = (a == b); // false
        boolean isNotEqual = (a != b); // true
        boolean isGreater = (a > b); // true
        boolean isLess = (a < b); // false
        boolean isGreaterOrEqual = (a >= b); // true
        boolean isLessOrEqual = (a <= b); // false

Example of logical operators:


        boolean isAdult = true;
        boolean hasPermission = false;
        boolean canEnter = isAdult && hasPermission; // false
        boolean isAdminOrUser = isAdult || hasPermission; // true
        boolean isNotAdmin = !hasPermission; // true

Example of assignment operators:


        int x = 10;
        x += 5; // x = x + 5, result: 15
        x -= 2; // x = x - 2, result: 13
        x *= 3; // x = x * 3, result: 39
        x /= 3; // x = x / 3, result: 13
        x %= 4; // x = x % 4, result: 1

Example of bitwise operators:


        int c = 5;  // 0101 in binary
        int d = 3;  // 0011 in binary
        int bitwiseAnd = c & d; // 0001 (1 in decimal)
        int bitwiseOr = c | d; // 0111 (7 in decimal)
        int bitwiseXor = c ^ d; // 0110 (6 in decimal)
        int bitwiseComplement = ~c; // 1010 (in two's complement, -6 in decimal)

Example of increment and decrement operators:


        int y = 10;
        y++; // Post-increment, y becomes 11
        ++y; // Pre-increment, y becomes 12
        y--; // Post-decrement, y becomes 11
        --y; // Pre-decrement, y becomes 10

13. Java If-Else

In Java, the if-else statement is used to execute code based on certain conditions. It allows you to control the flow of a program by running specific blocks of code only if certain conditions are met

if statement: Executes a block of code if the condition is true.

else statement: Executes a block of code if the condition is false.

else if statement: Used for multiple conditions, checked in sequence.

Example of using an if-else statement:


          int number = 10;
          if (number > 0) {
              System.out.println("Positive number");
          } else {
              System.out.println("Not a positive number");
          }

Another example:

One more example:

14. Java Switch

The switch statement in Java is used to execute one block of code among many based on the value of a variable.

the switch statement provides a way to execute different parts of code based on the value of a variable. It is a cleaner alternative to using multiple if-else statements, especially when dealing with several possible values for a single variable.

Switch Statement: A control statement that allows multi-way branching based on the value of an expression

Cases: Each case corresponds to a potential value of the expression.

Break Statement: Used to exit the switch after a case has been executed

Default Case: Optional and executes if no cases match the expression

Fall-Through Behavior: Omitting break allows execution to continue to subsequent cases.

Example of a switch statement:


          int day = 3;
          switch (day) {
              case 1:
                  System.out.println("Monday");
                  break;
              case 2:
                  System.out.println("Tuesday");
                  break;
              case 3:
                  System.out.println("Wednesday");
                  break;
              default:
                  System.out.println("Not a valid day");
          }

int day = 3;

int: This is a data type in Java used to store whole numbers (without decimal points). Here, it’s used to represent the day of the week as a number.

day: This is the variable name representing the day of the week. It’s assigned the value 3 in this example.

switch (day) { ... }

switch: This statement is used to perform different actions based on the value of a variable. It evaluates the variable day and checks each case to find a match.

case 1: When day is 1, this block runs.

System.out.println("Monday");: This prints "Monday" to the console if day equals 1.

break;: This statement exits the switch block, preventing the code from running into the next case.

case 2: When day is 2, this block runs.

System.out.println("Tuesday");: This prints "Tuesday" to the console if day equals 2.

break;: This exits the switch block.

case 3: When day is 3, this block runs.

System.out.println("Wednesday");: This prints "Wednesday" to the console if day equals 3.

break;: This exits the switch block.

default: This block runs if none of the case values match the day variable. It’s like a "catch-all" for unexpected values.

System.out.println("Not a valid day");: This prints "Not a valid day" to the console if day does not match any of the cases.

This switch structure allows for easy branching based on the value of day. In this example, because day is 3, it will print "Wednesday".

Another example:


          char grade = 'A';
          switch (grade) {
              case 'A':
                  System.out.println("Excellent");
                  break;
              case 'B':
                  System.out.println("Good");
                  break;
              default:
                  System.out.println("Grade not recognized");
          }

One more example:


          int month = 5;
          switch (month) {
              case 1:
                  System.out.println("January");
                  break;
              case 2:
                  System.out.println("February");
                  break;
              case 5:
                  System.out.println("May");
                  break;
              default:
                  System.out.println("Not a valid month");
          }

15. Java While Loop

The while loop in Java repeatedly executes a block of statements as long as the specified condition is true.

Example of a while loop:


          int count = 1;
          while (count <= 5) {
              System.out.println(count);
              count++;
          }

int count = 1;

int: This is a data type in Java used to store whole numbers (without decimal points). Here, it’s used to represent a counter.

count: This is the variable name. It starts with the value 1 and will increase with each iteration of the loop.

while (count <= 5) { ... }

while: This is a loop that repeats as long as the specified condition is true. In this case, the loop continues as long as count <= 5 (meaning count is less than or equal to 5).

System.out.println(count);: This line prints the current value of count to the console each time the loop runs.

count++;: This is shorthand for count = count + 1. It increases the value of count by 1 with each iteration of the loop.

So, this code starts with count at 1 and prints each number up to 5. When count reaches 6, the while loop condition is no longer true, and the loop stops.

Another example:


          int n = 10;
          while (n >= 1) {
              System.out.println(n);
              n--;
          }

One more example:


          int sum = 0;
          int i = 1;
          while (i <= 10) {
              sum += i;
              i++;
          }
          System.out.println("Total sum is: " + sum);

16. Java For Loop

The for loop in Java is used to execute a block of code a specific number of times.

Example of a for loop:


          for (int i = 1; i <= 5; i++) {
              System.out.println(i);
              //i++ increment
          }

for (int i = 1; i <= 5; i++) { ... }

for: This is a loop that repeats a block of code a set number of times. It’s commonly used when you know in advance how many times you want the loop to run.

int i = 1: This part initializes the loop variable i and sets it to 1 at the start.

i <= 5: This is the loop condition. The loop will continue to run as long as i is less than or equal to 5.

i++: This part increments i by 1 after each iteration. It’s shorthand for i = i + 1.

System.out.println(i);: This line prints the current value of i to the console each time the loop runs.

So, this code starts with i at 1 and prints each number up to 5. After i becomes 6, the condition i <= 5 is no longer true, and the loop stops.

Another example:


          for (int j = 10; j >= 1; j--) {
              System.out.println(j);
              //j-- decrement
          }

One more example:


          for (int k = 1; k <= 10; k++) {
              System.out.print(k + ", ");
          }

17. Java Break and Continue

The break statement is used to exit a loop, while the continue statement skips the current iteration and moves to the next one.

Example of using break:


          for (int i = 1; i <= 5; i++) {
              if (i == 3) {
                  break; // Exit the loop when i is 3
              }
              System.out.println(i);
          }

Example of using continue:


          for (int j = 1; j <= 5; j++) {
              if (j == 3) {
                  continue; // Skip the iteration when j is 3
              }
              System.out.println(j);
          }

One more example:


          for (int k = 1; k <= 5; k++) {
              if (k % 2 == 0) {
                  continue; // Skip even numbers
              }
              System.out.println(k);
          }

18. Java Arrays

Arrays in Java are used to store multiple values of the same type in a single variable.

Example of declaring and initializing a 1D array:


        int[] numbers = {1, 2, 3};

This creates an integer array called numbers with values 1, 2, 3.

Accessing array elements:


        int firstNumber = numbers[0]; // 1

This accesses the first element in the numbers array, which is 1.

Looping through an array:


        for (int i = 0; i < numbers.length; i++) {
            System.out.println(numbers[i]);
        }

This for loop prints all elements in the numbers array sequentially.

2D Arrays in Java

2D arrays in Java are arrays of arrays. They allow you to represent a matrix-like structure.

Example of declaring and initializing a 2D array:


        int[][] matrix = {
            {1, 2, 3},
            {4, 5, 6},
            {7, 8, 9}
        };

This creates a 3x3 matrix where:

  • Row 0: {1, 2, 3}
  • Row 1: {4, 5, 6}
  • Row 2: {7, 8, 9}

Accessing elements in a 2D array:


        int firstElement = matrix[0][0]; // 1
        int middleElement = matrix[1][1]; // 5

The element at matrix[0][0] is 1, and the element at matrix[1][1] is 5.

Looping through a 2D array:


        for (int i = 0; i < matrix.length; i++) {
            for (int j = 0; j < matrix[i].length; j++) {
                System.out.print(matrix[i][j] + " ");
            }
            System.out.println(); // Move to the next row
        }

This nested loop prints all elements in the 2D array, row by row.

Output for the above example:


        1 2 3
        4 5 6
        7 8 9
