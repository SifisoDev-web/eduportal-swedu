1. Python Introduction

Python is a popular programming language known for its easy readability and versatility. It is widely used in web development, data analysis, artificial intelligence, and more.

2. Getting Started with Python

To start using Python, you need to install it on your computer. You can download it from the official Python website.

3. Python Syntax

In Python, the syntax refers to the set of rules that defines how a Python program is written and interpreted. Unlike Java, Python uses indentation to define code blocks instead of curly braces, making it essential to follow the indentation rules precisely. Python syntax is designed to be clear and readable.

Example of basic syntax:


            print("Hello, Python!")  # This prints a string to the console

Example of a simple function in Python:


            def greet(name):
                print(f"Hello, {name}!")

            greet("Alice")  # Calls the function with "Alice" as the argument

Example of using a conditional statement:


            age = 18
            if age >= 18:
                print("You are an adult.")
            else:
                print("You are not an adult.")

4. Python Comments

In Python, comments are used to explain code and make it more readable. Comments are ignored by the Python interpreter and are only meant for human readers. They can be single-line comments or multi-line comments.

Example of a single-line comment:


            # This is a single-line comment
            print("Hello, Python!")  # This prints a greeting

Example of a multi-line comment using triple quotes:


            """
            This is a multi-line comment.
            It can span multiple lines.
            Useful for explaining larger code blocks.
            """
            print("Hello, World!")

Another way to write multi-line comments (multiple single-line comments):


            # This is a comment
            # that spans multiple lines
            print("Goodbye, World!")

5. Python Variables

In Python, variables are used to store data values. A variable is created the moment you assign a value to it. Unlike some languages, Python variables do not require explicit declaration of their type â€“ the type is inferred from the value assigned to the variable.

Example of defining variables:


            name = "Alice"  # String variable
            age = 25         # Integer variable
            is_student = True  # Boolean variable

Example of updating a variable:


            age = age + 1  # Updates age by adding 1

Example of multiple variable assignments:


            x, y, z = 5, 10, 15  # Assigns 5 to x, 10 to y, and 15 to z

Example of using variables in an expression:


            total_score = x + y + z  # Calculates the sum of x, y, and z
            print(total_score)  # Outputs the total score

6. Python Data Types

In Python, data types are used to define the type of a variable, which determines what kind of operations can be performed on it. Python has several built-in data types, including numeric types, sequences, mappings, and more.

Common data types in Python:

  • Integer: Whole numbers, e.g., 5, -10.
  • Float: Decimal numbers, e.g., 3.14, -0.001.
  • String: A sequence of characters, enclosed in single or double quotes, e.g., "Hello", 'World'.
  • Boolean: Represents one of two values: True or False.
  • List: An ordered collection of items, which can be of different types, e.g., [1, 2, 3], ["apple", "banana", "cherry"].
  • Tuple: An ordered, immutable collection of items, e.g., (1, 2, 3), ("red", "green", "blue").
  • Dictionary: A collection of key-value pairs, e.g., {"name": "Alice", "age": 25}.

Example of variable assignments with different data types:


            integer_var = 42                   # Integer
            float_var = 3.14159            # Float
            string_var = "Hello, Python!"   # String
            boolean_var = False              # Boolean
            list_var = [1, 2, 3]     # List
            tuple_var = (1, 2, 3)     # Tuple
            dict_var = {"key1": "value1", "key2": 2}  # Dictionary

Example of checking data types:


            print(type(integer_var))  # Outputs: <class 'int'>
            print(type(list_var))     # Outputs: <class 'list'>

7. Python Numbers

In Python, numbers are used for mathematical operations and are classified into three main types: integers, floats, and complex numbers.

  • Integer: Whole numbers, positive or negative, without decimals, e.g., 10, -3, 0.
  • Float: Numbers with decimals, e.g., 3.14, -0.5, 2.0.
  • Complex: Numbers with a real and an imaginary part, e.g., 3 + 4j, where j is the imaginary unit.

Example of creating different types of numbers:


            integer_num = 42         # Integer
            float_num = 3.14159    # Float
            complex_num = 2 + 3j    # Complex number

Example of basic arithmetic operations:


            sum_result = 5 + 3         # Addition (8)
            diff_result = 10 - 7      # Subtraction (3)
            prod_result = 4 * 2       # Multiplication (8)
            div_result = 15 / 3      # Division (5.0)
            int_div_result = 15 // 4  # Integer division (3)
            mod_result = 15 % 4      # Modulus (3)
            exp_result = 2 ** 3      # Exponentiation (8)

Example of using built-in functions for numbers:


            abs_value = abs(-7.5)            # Absolute value (7.5)
            rounded_value = round(3.14159, 2)  # Rounds to 2 decimal places (3.14)
            power_result = pow(2, 3)         # Raises 2 to the power of 3 (8)

8. Python Casting

In Python, casting is used to change the data type of a variable. This is useful when you need to convert values for mathematical operations or format data. Python provides several built-in functions for casting between common types.

Common casting functions in Python:

  • int(): Converts a value to an integer, removing decimals if present.
  • float(): Converts a value to a float, adding decimals if necessary.
  • str(): Converts a value to a string.

Example of casting to an integer:


            float_value = 3.8
            int_value = int(float_value)  # Converts 3.8 to 3

Example of casting to a float:


            int_value = 5
            float_value = float(int_value)  # Converts 5 to 5.0

Example of casting to a string:


            number = 10
            text = str(number)  # Converts 10 to "10"

Example of using casting in mathematical operations:


            a = "3"   # a is a string
            b = "4.5" # b is also a string

            sum_result = int(a) + float(b)  # Converts a and b to numbers and adds them (3 + 4.5 = 7.5)

9. Python Strings

In Python, strings are sequences of characters enclosed within single, double, or triple quotes. Strings are commonly used to handle text and provide many useful methods for manipulation.

Examples of defining strings:


            single_quote_str = 'Hello'         # Using single quotes
            double_quote_str = "World"         # Using double quotes
            triple_quote_str = """Hello, World!"""  # Using triple quotes for multi-line strings

Example of accessing characters in a string (indexing):


            greeting = "Hello"
            first_char = greeting[0]  # Accesses 'H'
            last_char = greeting[-1]  # Accesses 'o'

Example of string concatenation:


            part1 = "Hello, "
            part2 = "Python!"
            message = part1 + part2  # Concatenates to "Hello, Python!"

Example of common string methods:


            text = "  Python Programming  "

            lower_text = text.lower()      # Converts to lowercase: "  python programming  "
            upper_text = text.upper()      # Converts to uppercase: "  PYTHON PROGRAMMING  "
            stripped_text = text.strip()   # Removes leading/trailing spaces: "Python Programming"
            replaced_text = text.replace("Python", "Java")  # Replaces "Python" with "Java": "  Java Programming  "

Example of using formatted strings:


            name = "Alice"
            age = 25
            greeting = f"Hello, {name}! You are {age} years old."  # f-string format: "Hello, Alice! You are 25 years old."

10. Python Booleans

In Python, a boolean is a data type that represents one of two values: True or False. Booleans are often used in conditional statements and loops to control the flow of a program based on specific conditions.

Examples of boolean values:


            is_sunny = True
            is_raining = False

Example of a boolean expression:


            x = 5
            y = 10
            result = x < y  # Evaluates to True

Example of using booleans in an if statement:


            is_adult = True
            if is_adult:
                print("Access granted")
            else:
                print("Access denied")

Example of boolean operators:

  • and: Returns True if both conditions are true.
  • or: Returns True if at least one condition is true.
  • not: Inverts the boolean value.

            a = True
            b = False

            and_result = a and b  # False
            or_result = a or b   # True
            not_result = not a   # False

11. Python Operators

In Python, operators are special symbols that perform operations on variables and values. They are categorized into several types based on their functionality:

  • Arithmetic Operators: Used for mathematical calculations.
  • Comparison Operators: Used to compare values.
  • Logical Operators: Used to combine conditional statements.
  • Assignment Operators: Used to assign values to variables.
  • Bitwise Operators: Used to perform operations on bits.
  • Membership Operators: Used to test for membership in sequences.
  • Identity Operators: Used to compare the memory locations of two objects.

1. Arithmetic Operators

Examples of arithmetic operators:


              a = 10
              b = 3

              sum_result = a + b         # Addition (13)
              diff_result = a - b      # Subtraction (7)
              prod_result = a * b       # Multiplication (30)
              div_result = a / b      # Division (3.333...)
              mod_result = a % b       # Modulus (1)
              exp_result = a ** b      # Exponentiation (1000)
              int_div_result = a // b  # Integer Division (3)

2. Comparison Operators

Examples of comparison operators:


              x = 5
              y = 10

              equals = x == y        # Equal (False)
              not_equals = x != y    # Not equal (True)
              greater_than = x > y       # Greater than (False)
              less_than = x < y        # Less than (True)
              greater_or_equal = x >= y # Greater than or equal (False)
              less_or_equal = x <= y  # Less than or equal (True)

3. Logical Operators

Examples of logical operators:


              a = True
              b = False

              and_result = a and b  # False
              or_result = a or b   # True
              not_result = not a   # False

4. Assignment Operators

Examples of assignment operators:


              c = 5                # Simple assignment
              c += 2               # c = c + 2 (7)
              c -= 1               # c = c - 1 (6)
              c *= 3               # c = c * 3 (18)
              c /= 2               # c = c / 2 (9.0)

5. Bitwise Operators

Examples of bitwise operators:


              x = 5   # Binary: 0101
              y = 3   # Binary: 0011

              and_result = x & y    # Bitwise AND (1)
              or_result = x | y     # Bitwise OR (7)
              xor_result = x ^ y     # Bitwise XOR (6)
              not_result = ~x                # Bitwise NOT (-6)
              left_shift = x << 1    # Left shift (10)
              right_shift = x >> 1   # Right shift (2)

6. Membership Operators

Examples of membership operators:


              my_list = [1, 2, 3]
              is_present = 2 in my_list   # True
              is_not_present = 4 not in my_list # True

7. Identity Operators

Examples of identity operators:


              a = 5
              b = 5
              is_same = a is b         # True (same value)
              is_not_same = a is not b # False

12. Python Lists

A list in Python is a collection data type that is ordered and mutable. It can hold a variety of data types, including numbers, strings, and other lists. Lists are defined using square brackets [].

Creating a list:


            fruits = ['apple', 'banana', 'cherry']

Accessing List Elements

List elements can be accessed using their index, with the first element having an index of 0.


            first_fruit = fruits[0]  # 'apple'
            second_fruit = fruits[1]  # 'banana'

Modifying List Elements

Lists are mutable, which means you can change their content:


            fruits[1] = 'blueberry'  # Changing 'banana' to 'blueberry'

List Methods

Python provides several built-in methods to manipulate lists:

  • append(value): Adds an item to the end of the list.
  • remove(value): Removes the first occurrence of a value.
  • insert(index, value): Inserts a value at a specified index.
  • pop(index): Removes and returns the item at the specified index.
  • sort(): Sorts the list in ascending order.
  • reverse(): Reverses the order of the list.

Example of using list methods:


            fruits.append('orange')       # Adds 'orange'
            fruits.remove('apple')       # Removes 'apple'
            fruits.insert(1, 'kiwi')  # Inserts 'kiwi' at index 1
            last_fruit = fruits.pop()  # Removes and returns 'orange'
            fruits.sort()  # Sorts the list
            fruits.reverse()  # Reverses the list

List Comprehensions

List comprehensions provide a concise way to create lists:


            squares = [x ** 2 for x in range(10)]  # [0, 1, 4, 9, 16, 25, 36, 49, 64, 81]

Nested Lists

Lists can contain other lists, allowing for multidimensional arrays:


            matrix = [[1, 2], [3, 4]]
            element = matrix[1][0]  # 3

13. Python Tuples

A tuple in Python is an immutable collection data type that is ordered and can hold a variety of data types. Tuples are defined using parentheses ().

Creating a tuple:


            colors = ('red', 'green', 'blue')

Accessing Tuple Elements

Elements in a tuple can be accessed using their index, similar to lists:


            first_color = colors[0]  # 'red'
            second_color = colors[1]  # 'green'

Tuple Operations

Tuples support various operations:

  • Concatenation: Joining two tuples together.
  • Repetition: Repeating a tuple multiple times.
  • Membership: Checking if an item exists in a tuple.

Examples:


            more_colors = ('yellow', 'purple')
            all_colors = colors + more_colors  # Concatenation
            repeated_colors = colors * 2  # Repetition
            is_red_present = 'red' in colors  # True

Immutability

Tuples are immutable, meaning once they are created, their contents cannot be changed:


            # This will raise an error:
            colors[0] = 'orange'  # TypeError

Tuple Packing and Unpacking

Tuples can be packed and unpacked easily:


            point = (10, 20)  # Packing
            x, y = point  # Unpacking

Using Tuples as Dictionary Keys

Because tuples are immutable, they can be used as keys in dictionaries:


            coordinates = {(1, 2): 'A', (3, 4): 'B'}
            value = coordinates[(1, 2)]  # 'A'

14. Python Sets

A set in Python is an unordered collection of unique items. Sets are mutable, meaning you can add or remove elements after creation, and they are defined using curly braces {} or the set() constructor.

Creating a set:


            fruits = {'apple', 'banana', 'cherry'}

Accessing Sets

Since sets are unordered, they do not support indexing or slicing. You can check for membership using the in keyword:


            is_banana_present = 'banana' in fruits  # True

Modifying Sets

Sets allow you to add or remove elements:


            fruits.add('orange')  # Adds 'orange'
            fruits.remove('banana')  # Removes 'banana'

Set Methods

Python provides several built-in methods to manipulate sets:

  • union(other_set): Returns a new set with all items from both sets.
  • intersection(other_set): Returns a new set with items common to both sets.
  • difference(other_set): Returns a new set with items in the first set but not in the second.
  • symmetric_difference(other_set): Returns a new set with items in either set but not in both.
  • clear(): Removes all items from the set.

Examples of Set Methods:


            vegetables = {'carrot', 'potato'}
            all_items = fruits.union(vegetables)  # Union
            common_items = fruits.intersection(vegetables)  # Intersection
            only_fruits = fruits.difference(vegetables)  # Difference
            unique_items = fruits.symmetric_difference(vegetables)  # Symmetric Difference

Creating Sets from Other Data Types

You can create a set from a list, string, or tuple, which automatically removes duplicates:


            numbers_list = [1, 2, 2, 3]
            unique_numbers = set(numbers_list)  # {1, 2, 3}

Frozensets

A frozenset is an immutable version of a set, meaning its contents cannot be changed after creation:


            frozen_fruits = frozenset(fruits)

15. Python Dictionaries

A dictionary in Python is an unordered collection of key-value pairs. Each key must be unique and immutable, while the value can be of any data type. Dictionaries are defined using curly braces {} with a colon : separating keys and values.

Creating a dictionary:


            student = {
                'name': 'Alice',
                'age': 20,
                'major': 'Computer Science'
            }

Accessing Dictionary Values

Values in a dictionary can be accessed using their keys:


            student_name = student['name']  # 'Alice'

Modifying Dictionaries

You can add new key-value pairs or modify existing ones:


            student['graduation_year'] = 2024  # Adding a new key
            student['age'] = 21  # Modifying existing key

Removing Key-Value Pairs

Items can be removed using the del statement or the pop() method:


            del student['major']  # Removes the key 'major'
            age = student.pop('age')  # Removes and returns the value of 'age'

Dictionary Methods

Python provides several built-in methods to manipulate dictionaries:

  • keys(): Returns a view of all keys in the dictionary.
  • values(): Returns a view of all values in the dictionary.
  • items(): Returns a view of all key-value pairs as tuples.
  • clear(): Removes all items from the dictionary.

Examples of Dictionary Methods:


            student_keys = student.keys()  # dict_keys(['name', 'graduation_year'])
            student_values = student.values()  # dict_values(['Alice', 2024])
            student_items = student.items()  # dict_items([('name', 'Alice'), ('graduation_year', 2024)])
            student.clear()  # Removes all items

Nesting Dictionaries

Dictionaries can contain other dictionaries, allowing for complex data structures:


            students = {
                'Alice': {'age': 20, 'major': 'Computer Science'},
                'Bob': {'age': 22, 'major': 'Mathematics'}
            }
            bob_major = students['Bob']['major']  # 'Mathematics'

16. Python If-Else Statements

The if-else statement in Python is used for conditional execution of code blocks. It allows you to execute certain code only when specific conditions are met.

Basic If Statement

The simplest form of an if statement checks a condition and executes a block of code if the condition is true:


        age = 18

        if age >= 18:
            print("You are an adult.")

If-Else Statement

To execute alternative code when the condition is false, use the else clause:


        if age >= 18:
            print("You are an adult.")
        else:
            print("You are a minor.")

If-Elif-Else Ladder

You can chain multiple conditions using elif (short for "else if"):


        score = 85

        if score >= 90:
            print("Grade: A")
        elif score >= 80:
            print("Grade: B")
        elif score >= 70:
            print("Grade: C")
        else:
            print("Grade: D")

Nested If Statements

You can also nest if statements inside another if statement:


        number = 10

        if number > 0:
            print("Positive number")
            if number % 2 == 0:
                print("Even number")
            else:
                print("Odd number")
        else:
            print("Negative number")

Short-Circuiting with Logical Operators

You can combine conditions using logical operators and, or, and not:


        temperature = 30

        if temperature > 25 and temperature < 35:
            print("Warm day")
        else:
            print("Not a warm day")

17. Python While Loop

A while loop in Python repeatedly executes a block of code as long as a specified condition is true. It is useful for situations where the number of iterations is not known beforehand.

Basic While Loop

Hereâ€™s a simple example of a while loop that counts from 1 to 5:


            count = 1

            while count <= 5:
                print(count)
                count += 1  # Increment count

Breaking Out of a Loop

You can exit a while loop prematurely using the break statement:


            count = 1

            while count <= 10:
                if count == 6:
                    break  # Exit the loop when count is 6
                print(count)
                count += 1

Skipping Iterations

The continue statement skips the current iteration and moves to the next one:


            count = 0

            while count < 5:
                count += 1
                if count == 3:
                    continue  # Skip the print statement when count is 3
                print(count)

Using While Loop with Lists

A while loop can also be used to iterate over a list:


            fruits = ['apple', 'banana', 'cherry']
            index = 0

            while index < len(fruits):
                print(fruits[index])
                index += 1

Infinite Loops

Be cautious of infinite loops, which occur when the condition of the while loop always evaluates to true:


            count = 1

            while count > 0:  # This loop will run indefinitely
                print(count)

To avoid infinite loops, ensure that the loop condition will eventually evaluate to false.

18. Python For Loop

A for loop in Python is used to iterate over a sequence (like a list, tuple, dictionary, set, or string). It allows you to execute a block of code for each item in the sequence.

Basic For Loop

Hereâ€™s a simple example of a for loop that iterates through a list of fruits:


            fruits = ['apple', 'banana', 'cherry']

            for fruit in fruits:
                print(fruit)

Using the Range Function

The range() function generates a sequence of numbers, which can be used in a for loop:


            for i in range(5):  # 0 to 4
                print(i)

For Loop with Index

You can also access the index of items using the enumerate() function:


            fruits = ['apple', 'banana', 'cherry']

            for index, fruit in enumerate(fruits):
                print(index, fruit)

Nested For Loops

You can nest for loops to iterate over multiple sequences:


            colors = ['red', 'green', 'blue']
            fruits = ['apple', 'banana']

            for color in colors:
                for fruit in fruits:
                    print(color, fruit)

List Comprehensions

For loops can be simplified using list comprehensions, allowing you to create lists in a concise manner:


            squares = [x**2 for x in range(10)]
            print(squares)  # Output: [0, 1, 4, 9, 16, 25, 36, 49, 64, 81]

Exiting a For Loop

Similar to while loops, you can use the break statement to exit a for loop:


            for number in range(10'):
                if number == 5:
                    break  # Exit the loop when number is 5
                print(number)

19. Python Functions

A function in Python is a block of reusable code that performs a specific task. Functions help organize code, make it more readable, and facilitate code reuse.

Defining a Function

You define a function using the def keyword, followed by the function name and parentheses:


            def greet(name):
                print('Hello, ' + name + '!')

Calling a Function

To execute the code inside a function, you call it by its name and pass any required arguments:


            greet('Alice')  # Output: Hello, Alice!

Function with Return Value

A function can return a value using the return statement:


            def add(a, b):
                return a + b

            result = add(5, 3)
            print(result)  # Output: 8

Default Parameter Values

You can provide default values for function parameters:


            def power(base, exponent = 2):
                return base ** exponent

            print(power(4))          # Output: 16
            print(power(3, 3))  # Output: 27

Keyword Arguments

You can call functions using keyword arguments for clarity:


            def describe_pet(animal_type, pet_name):
                print('I have a ' + animal_type + ' named ' + pet_name + '!')

            describe_pet('hamster', 'Harry')  # Positional arguments
            describe_pet(pet_name='Willie', animal_type='dog')  # Keyword arguments

Lambda Functions

Python also supports anonymous functions, called lambda functions, which can take any number of arguments but only have one expression:


            multiply = lambda x, y: x * y
            print(multiply(4, 5))  # Output: 20

20. Python Lambda Functions

A lambda function in Python is a small anonymous function that can take any number of arguments but can only have one expression. Lambda functions are often used for short, throwaway functions that are not bound to a name.

Defining a Lambda Function

You can define a lambda function using the lambda keyword, followed by the parameters, a colon, and the expression:


            square = lambda x: x ** 2

Using a Lambda Function

You can call a lambda function like any regular function:


            print(square(5))  # Output: 25

Lambda Functions with Multiple Arguments

Lambda functions can accept multiple arguments:


            add = lambda a, b: a + b
            print(add(3, 4))  # Output: 7

Using Lambda with Built-in Functions

Lambda functions are often used with built-in functions like map(), filter(), and sorted():

Map Example


            numbers = [2, 4]
            squared = list(map(lambda x: x ** numbers))
            print(squared)  # Output: [1, 4, 9, 16]

Filter Example


            even_numbers = list(filter(lambda x: x % 0, numbers))
            print(even_numbers)  # Output: [2, 4]

Sorted Example


            points = [(1, 2), (3, 1), (5, 0)]
            sorted_points = sorted(points, key=lambda point: point[1])
            print(sorted_points)  # Output: [(5, 0), (3, 1), (1, 2)]

When to Use Lambda Functions

Lambda functions are ideal for short, simple operations where defining a full function might be overkill. They are commonly used for:

  • Simple transformations of data
  • Functional programming techniques
  • Callbacks in higher-order functions

21. Python Arrays

In Python, an array is a collection of items stored at contiguous memory locations. They are similar to lists, but with some key differences. The main difference is that arrays can only contain elements of the same data type.

Using the Array Module

To work with arrays, you can use the array module, which provides a more efficient way of storing data than lists:


            import array as arr

            numbers = arr.array('i', [1, 2, 3, 4])

Array Types

The first argument to the array constructor is a type code, which specifies the type of elements in the array. Some common type codes include:

  • 'i' - signed integer
  • 'f' - floating point
  • 'd' - double precision float
  • 'b' - signed byte

Accessing Array Elements

You can access elements in an array using indexing, similar to lists:


            print(numbers[0])  # Output: 1

Modifying Array Elements

Array elements can be modified by assigning new values:


            numbers[1] = 10
            print(numbers)  # Output: array('i', [1, 10, 3, 4])

Array Methods

Arrays come with several useful methods:

Append Method


            numbers.append(5)
            print(numbers)  # Output: array('i', [1, 10, 3, 4, 5])

Insert Method


            numbers.insert(2, 20)
            print(numbers)  # Output: array('i', [1, 10, 20, 3, 4, 5])

Remove Method


            numbers.remove(10)
            print(numbers)  # Output: array('i', [1, 20, 3, 4, 5])

Iterating Over an Array

You can iterate over the elements of an array using a for loop:


            for num in numbers:
                print(num)

Converting Lists to Arrays

24. Python Iterators

An iterator is an object that allows you to traverse through a collection, such as a list, tuple, or dictionary, without the need for an index variable. In Python, iterators implement two special methods: __iter__() and __next__().

To create an iterator, you need to define a class that implements the __iter__() and __next__() methods:

                    self.end = end

                    return self

                    if self.current self.end:
                        value = self.current
                        self.current += 1
                        return value
                    else:
                        raise StopIteration

Once you have defined an iterator, you can use it in a loop:

            print(next(my_iterator))  # Output: 3

Handling StopIteration

When the iterator has no more elements to return, it raises a StopIteration exception. You can handle this exception using a try and except block:


            try:
                while True:
                    print(next(my_iterator))
            except StopIteration:
                pass

A simpler way to create iterators is by using generator functions. You can define a generator using the yield keyword:

26. Python Scope

Scope refers to the visibility of variables in a program. In Python, scope determines the area in which a variable can be accessed and modified. Understanding scope is crucial for writing clean and effective code.

Types of Scope

  • Local Scope: Variables declared within a function are local to that function and cannot be accessed outside of it.
  • Global Scope: Variables declared outside of all functions have a global scope and can be accessed anywhere in the code.
  • Enclosing Scope: Refers to the scope of enclosing functions. A nested function can access variables from its enclosing function.
  • Built-in Scope: This includes names pre-defined in Python, such as keywords and built-in functions (e.g., print(), len()).

LEGB Rule

Python uses the LEGB rule to resolve variable names:

  • L: Local â€” names assigned in the current function.
  • E: Enclosing â€” names in the local scope of enclosing functions.
  • G: Global â€” names assigned at the top-level of a module or declared global.
  • B: Built-in â€” names pre-defined in the built-in scope.

  • Be mindful of the enclosing scope when using nested functions.

31. Python Regex

Regular expressions (regex) are sequences of characters that define search patterns. In Python, the re module provides a powerful way to work with regular expressions, allowing you to search, match, and manipulate strings using pattern matching.

To use regular expressions in Python, you need to import the re module:

This function searches the entire string for a match:

            text = 'Hello, world!'

Regular expressions support special characters to enhance pattern matching:

  • . - Matches any character except a newline.
  • \d - Matches any digit (equivalent to [0-9]).
  • \D - Matches any non-digit character.
  • \w - Matches any alphanumeric character (equivalent to [a-zA-Z0-9_]).
  • \W - Matches any non-alphanumeric character.
  • \s - Matches any whitespace character (spaces, tabs, newlines).

You can define character classes using square brackets:

            text = 'Hello, world!'

Quantifiers

Quantifiers specify how many times a character or group should be matched:

  • * - Matches 0 or more occurrences.
  • + - Matches 1 or more occurrences.
  • ? - Matches 0 or 1 occurrence.
  • {n} - Matches exactly n occurrences.
  • {n,} - Matches n or more occurrences.
  • {n,m} - Matches between n and m occurrences.

Best Practices for Using Regex in Python

  • Use raw strings (prefix with r) for regex patterns to avoid issues with escape sequences.
  • Test your regex patterns thoroughly, as they can become complex and difficult to debug.
  • Use comments in your regex patterns to describe their purpose, especially for more complicated patterns.

f-Strings (Formatted String Literals)

Starting from Python 3.6, f-strings provide a more concise and readable way to format strings. You can prefix a string with f and include variables directly within curly braces:


            name = 'Bob'
            age = 25
            greeting = f'Hello, {name}. You are {age} years old.'
            print(greeting)

f-strings are not only easier to write but also more efficient than the format() method.

Using the % Operator

Another way to format strings is using the percent % operator. While this method is considered less modern, it is still widely used:


            name = 'Charlie'
            age = 28
            greeting =  'Hello, %s. You are %d years old.' % (name, age)
            print(greeting)

In this example, %s is a placeholder for a string, and %d is for an integer.

Controlling Formatting

You can control how values are formatted, such as specifying the number of decimal places for floating-point numbers:


            pi = 3.14159
            formatted_pi = '{:.2f}'.format(pi)
            print('Formatted pi: ' + formatted_pi)  # Output: Formatted pi: 3.14

In this case, {:.2f} indicates that pi should be formatted as a floating-point number with two decimal places.

Best Practices for String Formatting

  • Use f-strings for clarity and efficiency in Python 3.6 and later.
  • Keep your formatting consistent throughout your code to improve readability.
  • When dealing with multiple values, consider using the format() method or f-strings for better management of placeholders.
  • Be mindful of data types and use appropriate formatting specifiers.
